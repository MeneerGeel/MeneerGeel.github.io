Construction block - Iteration 1

I wanted to make something to come up with shapes for garments, and thought of several ways in which small pieces of fabric could be fitted together to produce a final shape (Not yet sure if this idea is more convienient than current approaches).

Some more basic answers to the question "how can we connect two solid blocks to each other"

An idea in between the fabric 'blocks' and the solid blocks: based on (starch) packing peanuts' ability to be stuck together. Perhaps this can be replicated by making a cilindrical block covered in velcro?

Construction block - Iteration 2

I liked the packing peanut idea, and decided to make it. I found sticky velcro patches, which I attached to used up toilet rolls. The velcro patches alternate (hook - loop - hook on one side, loop - hook - loop on the other) to create a lot of possible ways to attach the blocks to each other. I also incorporated a used kitchen towel roll, for more variety.

Although the idea is simple, these things were surprisingly easy and fun to mess around with! A little gallery:
